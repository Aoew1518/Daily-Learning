# 分饼干

- 最值问题
    不适合动态规划来解决
- 贪心算法  greedy
    - 局部最优解    当下这一步  最小尺寸的饼干只给最小胃口的孩子 +1
        最优子结构 和全局最优子结构是一致的 全国一盘棋
    
    - 全局最优解
    先满足贪心局部最优解
    再求全局最优解
    局部最优解不一定全局最优解
    贪心算法证明
        反证法
        数学归纳法
    贪心算法时间复杂度
        O(n)
- 回溯算法
    暴力穷举
    时间复杂度
        指数级别
    回溯算法
        剪枝
        时间复杂度
            O(n!)
- 动态规划
    最优子结构
    状态转移方程
    重叠子问题
    时间复杂度
        O(n^2)
