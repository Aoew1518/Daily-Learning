# 预编译头发生在函数体内时
1. 创建函数的AO对象（Action Object）
2. 找形参和变量声明，将形参和变量声明作为AO的属性名，值赋予undefined
3. 将形参和实参统一
4. 在函数体内找函数声明，将函数声明作为AO对象的属性名，值赋予函数体

# 预编译发生在全局
1. 创建GO对象 （Global Object）
2. 找变量声明，将变量声明作为GO的属性名，值赋予undefined
3. 在全局找函数声明，将函数声明作为GO的属性名，值赋予函数体
